LukatinCode

基于Go语言开发的Claude Code翻版，集成持久化Shell、代码检索与编辑、Todo管理等能力，支持多家大模型

持久化Shell
内置持久化Bash环境，跨命令保留状态，支持环境变量和路径持久化，适合构建和脚本执行
代码检索与编辑
集成Grep/Glob/LS等检索工具，支持Read/Edit/MultiEdit/Write等文件操作，实现精确的代码浏览与批量编辑
Todo管理系统
结构化待办清单管理，支持TodoRead/TodoWrite操作，强约束校验确保任务状态一致性
多模型支持
支持OpenAI、Anthropic、DeepSeek、Google、Qwen等多家大模型，通过配置文件灵活切换
TUI界面
基于Bubble Tea构建的现代终端用户界面，实时显示工具调用进度、TodoList状态和日志输出
联网搜索能力
集成WebFetch/WebSearch/Task等工具，支持网页分析、联网搜索和子Agent扩展搜索功能

快速开始

配置并运行LukatinCode：

# 1. 克隆项目 $ git clone https://github.com/ccIisIaIcat/FakeClaudeCode $ cd FakeClaudeCode # 2. 配置模型（编辑 LLMConfig.yaml） AgentAPIKey: OpenAI: BaseUrl: https://api.openai.com/v1 APIKey: sk-xxx Model: gpt-4o # 3. 运行程序 $ go run main.go

支持的模型

OpenAI (GPT-4, GPT-3.5等) Anthropic (Claude系列) DeepSeek (DeepSeek-V2等) Google (Gemini系列) Qwen (通义千问系列)

Docker使用

# Linux容器环境 $ docker compose up -d $ docker exec -it go-linux-dev bash $ cd /workspace && go run main.go

如何使用 LukatinCode（AI 助手）

LukatinCode 同时包含一个交互式 AI 助手（即本项目内置的智能代理），可以通过自然语言指令调用内置工具来完成代码检索、编辑与任务管理。

# 示例交互（终端或TUI中） User: 帮我查找所有包含 "TODO" 的文件并列出 Assistant: 正在使用 Grep 搜索... (返回文件列表) User: 请在 src/main.go 中把变量名 oldName 改为 newName Assistant: 已使用 Edit 操作修改 src/main.go User: 新增一个待办：添加单元测试 Assistant: 使用 TodoWrite 创建任务并返回任务ID

可用工具（示例）：

- Grep, Glob: 代码检索 - Read, Edit, MultiEdit, Write: 文件读写与批量编辑 - TodoRead, TodoWrite: 结构化任务管理 - WebFetch, WebSearch, Task: 联网与子Agent扩展

建议的最小工作流：

1) 使用 Grep/Glob 定位问题文件 2) 使用 Read 查看上下文 3) 使用 Edit/MultiEdit 进行修改并保存 4) 使用 TodoWrite 记录变更或后续任务
